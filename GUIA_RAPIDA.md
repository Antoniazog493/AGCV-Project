# ? GUÍA RÁPIDA - AGCV + BetterJoy

## ?? USO EN 3 PASOS

### 1?? Abre AGCV
```
Ejecuta: AGCV\bin\Debug\net8.0-windows\capaPresentacion.exe
```

### 2?? Inicia Sesión
```
Usuario: tu_usuario
Contraseña: tu_contraseña
```

### 3?? Conecta tu Joy-Con
```
Click en "Conectar Control" ? BetterJoy se abre ? ¡Listo!
```

---

## ?? SI BETTERJOY NO SE ENCUENTRA

1. El sistema te preguntará si quieres seleccionarlo manualmente
2. Navega a: `BetterJoyForCemu\bin\x64\Debug\net8.0-windows\`
3. Selecciona: `BetterJoyForCemu.exe`
4. La ruta se guardará para la próxima vez

---

## ?? BOTONES EN EL MENÚ PRINCIPAL

| Botón | Función |
|-------|---------|
| ?? **Conectar Control** | Abre BetterJoy |
| ?? **Estadísticas** | Ver estadísticas (placeholder) |
| ?? **Ajustes** | Abrir configuración |
| ?? **Cerrar Sesión** | Salir al login |

---

## ?? PROBLEMAS COMUNES

### ? "No se encontró BetterJoyForCemu.exe"
**Solución:** Compila BetterJoyForCemu primero o selecciónalo manualmente

### ? "No se pudo iniciar BetterJoy"
**Solución:** Ejecuta Visual Studio como Administrador

### ?? "BetterJoy ya está en ejecución"
**Solución:** No es un error, BetterJoy ya está abierto
